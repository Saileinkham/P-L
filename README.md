# OKJ P&L Dashboard

## โครงสร้างไฟล์
```
index.html       - หน้าหลัก
config.js        - MGR zones, USERS, branch mapping
app.js           - ฟังก์ชันทั้งหมด
css/app.css      - styles
data/2026-01/    - ข้อมูลเดือน ม.ค. 2026
  raw.json       - branch summaries
  dd.json        - P&L detail rows
  ic.json        - item cost
```
